At Pirum Systems, I work as a Data Engineer building the data backbone for a
fintech platform used by 150+ financial institutions globally, including 25 of
the 30 G-SIBs.

**Key contributions:**


- **ETL Pipeline Architecture:** Built and maintain 20+ ETL pipelines using AWS
  Glue and PySpark, processing 2.5TB of financial data daily through a
  medallion architecture (bronze/silver/gold layers).

- **Business-Critical Aggregations:** Designed silver-to-gold pipelines with
  complex business logic for client invoicing, ROI calculations, and KPI
  monitoring dashboards.

- **Infrastructure as Code:** Manage cloud infrastructure using Terraform
  across Glue, S3, Athena, Lambda, SNS, SQS, SageMaker, and Bedrock.

- **Cost Optimisation:** Reduced Glue job costs by 75% through partition
  pruning, predicate pushdown, and CloudWatch-informed Spark worker rightsizing.

**Tech stack:** AWS (Glue, S3, Athena, Lambda, Bedrock, SageMaker), PySpark,
Terraform, Python, SQL
